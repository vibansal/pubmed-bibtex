python script for BibTeX retrieval from PubMed and bioRxiv
- Fetch BibTeX by PMID or keyword query via PubMed NCBI API
- Search bioRxiv/medRxiv preprints via Europe PMC
- Keyword-based re-ranking of results (optional, --rank)
- Batch processing from input file (PMIDs or keyword queries auto-detected)
- CLI options: -i/--id, -q/--query, -K/--max-results, --source, --rank, --short-journal
